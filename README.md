#  Sistema de Controle de Estoque

Um sistema simples de gerenciamento de estoque desenvolvido em Python.

##  Funcionalidades

-  Cadastrar produtos
-  Consultar produtos cadastrados
-  Atualizar informações de um produto
-  Remover produtos do estoque
-  Listar todos os produtos
-  Armazenamento dos dados em arquivo

##  Tecnologias Utilizadas

- Python 3.14
- Programação Orientada a Objetos
- Manipulação de arquivos

##  Estrutura do Projeto

```text
Projeto_Estoque/
│── main.py          # Arquivo principal
│── menu.py          # Interface do menu
│── crud.py          # Operações CRUD
│── estoque.py       # Classe e lógica do estoque
│── util.py          # Funções auxiliares
│── arquivo.py       # Manipulação de arquivos
```

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/Projeto_Estoque.git
```

2. Acesse a pasta do projeto:

```bash
cd Projeto_Estoque
```

3. Execute o programa:

```bash
python main.py
```
